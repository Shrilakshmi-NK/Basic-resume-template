# Basic Resume Template

A clean and responsive resume website template created using HTML, CSS, SCSS, and JavaScript. This template can be used to showcase personal information, experience, skills, and projects in a modern layout.

## Features

- Responsive design for mobile and desktop
- Modular HTML structure with reusable components
- Styled using CSS and SCSS for easy customization
- JavaScript support for dynamic effects
- Easy to customize with your own content
